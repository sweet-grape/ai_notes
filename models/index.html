FLUX.1（写实/广告）

  • 分辨率 1024 起步；Steps 22–30；CFG 5–7；Karras 系列采样器。
  • 提示词以材质与光线为核心：rim light, studio softbox, realistic texture。
  • 放大路线：latent → 4x-UltraSharp/ESRGAN → 轻修复 → 锐化。

Playground v2.5（插画/创意）

  • 分辨率 896/1024；Steps 18–26；CFG 6–8；色彩描述尽量明确。
  • 与 Lineart/Canny 组合可控构图；LoRA 0.5–0.8 控风格。

AuraFlow / Kandinsky（艺术向）

  • 强调画面结构与配色，多用“三分法/黄金螺旋 + 主次色”去约束。
  • 建议英文主词 + 中文备注；失败率高的主题加参考图更稳。

Stable Video Diffusion（视频）

  • 输入分辨率 576–768；长度 24–48 帧；FPS 12–16。
  • 图像一致性优先：固定种子 + 降噪 0.45–0.6；可叠 Pose/Depth。
