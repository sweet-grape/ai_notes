Wan 系列教程（Wan2 / i2v）

主打中文友好与高观感出图，适合插画、国风、商业海报与 i2v 视频；同样建议用 ComfyUI 跑统一流程。

1. Wan 特点

  • 中文解析强、默认审美统一，提示词成本低。
  • 风格与卡通/插画友好，文生视频（i2v）可快速出片。

2. 快速参数

  • 文生图：分辨率 1024，Steps 16–24，CFG 4–6，dpmpp_sde_karras。
  • 图生图：denoise 0.35–0.6；保留结构时配合 Canny/Lineart。
  • 一致性：固定 Seed；必要时锁词序与语义块。

3. 中文提示词模板

主体 + 场景/环境 + 构图/镜头 + 光线 + 风格/材质 + 颜色氛围 + 细节程度
例：中近景人物，环形光，淡金暖色调，插画风格，衣纹与发丝细节清晰
负面：多余手指，低清，噪点，水印，文字

4. 常见工作流

  1. 文生图：Load Wan → 文本编码 → KSampler → VAE Decode → Save
  2. 图生图：Image → VAE Encode → KSampler（denoise）→ VAE Decode → Save
  3. 风格锁定：IP-Adapter 参考图 + LoRA（0.6–0.8）

5. i2v 视频（Wan2.1）

  • 分辨率 1024×576/1024×1024；帧数 48–64；FPS 12–16。
  • Steps 24，CFG 5；motion 0.4–0.55；denoise 0.5–0.6；固定全部 seed（含 temporal）。
  • 不稳时加 Pose/Lineart/Depth 任一作约束；动效词：trailing particles/glow。

6. ControlNet/参考图

  • OpenPose（姿态/分镜）、Canny/Lineart（边缘/轮廓）、Depth（空间）。
  • IP-Adapter：锁风格/角色，多参考图混合更稳。

7. 优化建议

  • 中文优先，短句分块，核心语义靠前；数字与数量加“exactly/仅/严格”约束。
  • 保持分辨率与采样器一致；需要高清时，采用放大器 + 轻修复流程。

8. FAQ

为什么数量类约束不稳定？

叠加“6-fold symmetry / clock positions”等几何描述，并用负面词排除 5/7/8 等数量；必要时用参考草图。
