Seedance 2.0 提示词技巧

把「写什么」想清楚，比堆关键词更重要：通用公式、避坑、分阶段练习与可复制模板

Seedance 2.0 是什么、本文怎么用

Seedance 2.0 是字节跳动发布的视频生成大模型方向产品之一，强调多模态参考（图 / 视频 / 音频等，视具体接入产品而定）、音画一体与较高可控性。你在即梦、剪映相关能力、火山引擎文档等渠道看到的「Seedance」即属同一家族，但界面按钮、是否支持首帧图、时长上限会随版本变化。

阅读约定：本文讲的是提示词思路与结构，不替代官方参数说明。若某条与你当前使用的 App 不一致，以产品内说明与更新日志为准。可与本站 运镜提示词、视频帧技巧 对照使用。

先建立判断：好提示词在解决哪几件事

视频模型一次要同时猜「画什么、怎么动、多快、什么光、什么镜头」。提示词的作用，是把这几件事排好优先级，并避免互相打架。

  1. 单一主角任务：一句话里先让人听懂「谁 / 在干什么」，再补环境。
  2. 动作可执行：用身体能做出来的小动作（转头、伸手、走路），比抽象形容词更稳。
  3. 节奏与幅度：偏「慢、顺、稳」的描述通常更容易连贯；大跳、猛跑更容易形变（各代模型程度不同）。
  4. 风格一句话说清：例如「纪实摄影自然光」「动画电影柔和色彩」，不要堆十个风格词。
  5. 镜头像导演口令：景别 + 运动方式写清楚（可抄本站运镜表），少用「好看」「震撼」空词。

通用公式（背这一条就够入门）

把提示词想成一句短剧本 + 一句摄影指示：

[主体与状态] + [关键动作（含速度与方向）] + [环境/时间] + [光线与色彩气质] + [镜头/运镜] + [情绪或叙事一句]

同一请求里，越靠前的信息越像「主命令」；后面是对画面气质与拍法的补充。

模块 写什么（通俗说） 反面例子
主体与状态 谁、穿什么、大致年龄感 / 物种 「一个人」——信息太少
关键动作 具体动词 + 慢/中速 + 方向（从左到右、向镜头走来） 「做动作」——模型不知道做哪种
环境 室内/户外、大致地点、天气或时段 「很美的地方」——无法还原
光线与色彩 侧光、逆光、阴天柔光、暖色调夜景等 「高级」——无法映射到像素
镜头 景别 + 运镜（与 运镜页 对齐） 「电影感」单独出现——太空泛
情绪一句 孤独、期待、紧张——只选一个主情绪 又欢快又悲伤——容易糊

怎样给提示词更容易出「最佳观感」

所谓「最佳」，通常是主体清晰 + 运动可信 + 风格统一 + 没有明显穿帮。可按下面顺序自检。

1. 用「要什么」代替「不要什么」

很多视频模型对「负面提示词」不敏感或行为与绘图模型不同。与其写「不要糊、不要变形」，不如写正面画质与稳定性描述，例如：画面清晰锐利、主体轮廓稳定、运动平滑、皮肤质感自然。

2. 动作写成「可排练」的指令

把「跳舞」改成「双脚交替轻踏地面，手臂随节奏小幅摆动」；把「打斗」改成「侧身闪避，右手格挡，镜头保持中景」——越能想象排练，越容易生成连贯。

3. 控制复杂度：一镜头一事

一次生成只强调一个核心变化：要么主打运镜，要么主打人物表情，要么主打环境氛围。什么都强调等于什么都不强调。

4. 先短后长、先静后动

若产品支持短时长试跑，先用较短片段验证主体与风格，再扩展时长或做续写。静物、空镜、单人小动作往往比多人复杂互动更容易稳。

5. 有参考图 / 参考视频时写清「它管什么」

若界面支持用 @ 或素材槽引用参考，请在提示里说明用途，例如：参考图仅锁定人物外观；参考视频仅模仿运镜节奏。具体语法以当前产品为准。

6. 合规与真实人物

勿尝试生成可识别的真实名人、侵权角色或违规内容；这既影响成功率，也可能无法通过审核。

常见误区（对照改一句就好）

误区 为什么容易翻车 改法
堆 SD 式权重 (xxx:1.4) 视频模型常按自然语言理解，权重语法可能无效或干扰 改成口语强调：「特写为主」「全程保持侧逆光」
一次写满十几种风格 风格词互相抵消，画面发灰发脏 只保留一个主风格 + 一个辅风格
动作又大又快又久 大幅运动更易形变、跳帧 拆成多段；每段动作幅度减半、写清「缓慢」
主体太多还要互动 多主体抢注意力，容易脸糊、手糊 减人数；或固定一人清晰，他人虚化/远景
只靠形容词 「唯美」「大片感」无法翻译为镜头 换成光线、景别、运镜、色彩的具体词

学习方案（4 周，每天约 30～45 分钟）

目标：从「能出片」到「能稳定复现自己想要的镜头」。每周保留同一套记录习惯：提示词原文、参考素材、时长、满意点、翻车点（一句话）。

  1. 第 1 周 · 空镜与静物：只练环境与光线，不写人物。每天 3 条，分别侧重「时间（晨/夜）」「天气」「室内窗外景」。学会用一句话定调。
  2. 第 2 周 · 单人小动作：同一人物外观描述复制使用，只改动作。练习「慢速、方向、停顿时长」。对照 运镜页 加 1 种运镜词。
  3. 第 3 周 · 镜头语言：固定剧情一句（如「等人」），改景别与运镜组合。每天 1 条成片 + 1 条对照实验（只改运镜或只改光线）。
  4. 第 4 周 · 复盘与模板库：整理自己最稳的「开头句式」和「画质句式」，做个人模板表；尝试把一段拆成 2～3 段生成再衔接（若产品支持），体会与 视频帧技巧 的关系。

学时不必贪多：每周能沉淀 3～5 条「可复用句式」，比刷几十条无效提示更有用。

模板示例（可直接改括号内容）

以下为中文自然语言示例；若你使用的版本对英文更敏感，可把镜头与风格句换成英文，主体描写保留中文亦可，以实测为准。

模板 A · 人物 + 情绪 + 缓慢动作

一位 [年龄/性别] 的 [职业或身份感穿着]，独自站在 [具体地点]，[缓慢的具体动作，带方向]。时间：[清晨/黄昏]，[光线描述]。镜头：[景别]，[缓慢推近/固定/横移之一]。整体情绪：[一种]。画面清晰，肤色自然，运动平滑。

模板 B · 行走 + 环境纵深

从 [起点] 向 [终点] 缓慢行走，步幅不大，双臂自然摆动。场景是 [城市/林间/走廊]，背景行人虚化。日光 [柔和/侧光]，色调 [偏暖/偏冷]。镜头：中景跟拍，保持主体在画面一侧三分之一处。纪实感，稳定不抖动。

模板 C · 产品展示（静物旋转感由镜头完成）

[产品名称] 置于 [材质台面]，环境极简。[慢速] 环绕运镜或缓慢推近，突出 [一个细节：logo/材质/按钮]。柔光箱效果，高光干净，阴影柔和。背景干净无杂物。画面锐利，边缘整齐。

模板 D · 对话戏（少人、信息集中）

室内 [餐厅/客厅]，[两人] 坐在桌子两侧。[A] 说一句话时眼神看向 [B]，[B] 点头回应。镜头：正反打中的 [过肩/侧脸特写]，节奏舒缓。暖色灯光，浅景深，背景虚化。对白口型大致同步即可，情绪：[克制/紧张选一]。

模板 E · 风光 + 大气变化

[山地/海岸/城市天际线] 远景，[云层移动/雾气缓慢流动] 可见。时间 [日出前/蓝调时刻]，色彩 [描述主色与对比]。镜头：缓慢航拍或高位固定，避免剧烈摇摆。画面通透，层次清晰。

模板 F · 夜景人像（光少而准）

夜晚 [街道/窗边]，主光来自 [霓虹/路灯/室内台灯]，面部有清晰明暗过渡。人物 [站立/靠墙]，微表情 [平静/思念]。镜头：中近景，浅景深，背景光斑圆形。运动极小，以眼神变化为主。噪点控制自然，不要过曝。

「最佳效果」自检清单（生成前扫一眼）

  • 是否只有一个核心主体任务？
  • 动作是否写得具体到动词，并带慢/中速？
  • 风格与光线是否不超过两种主倾向？
  • 镜头是否包含景别 + 运镜各至少一个词？
  • 是否用正面语句写了「清晰、稳定、自然」中的至少一条？
